## Descripción

Este repositorio contiene un trabajo práctico para la materia Emprendedurismo.

La actividad consiste en:

1. Ver el documental **"El dilema de las redes sociales"**.
2. Realizar una reflexión personal sobre los aspectos del documental que se relacionan con la propia vida y experiencia.
3. Elaborar un archivo de texto con dicha reflexión.
4. Publicar el trabajo en un repositorio de GitHub y compartirlo a través de Moodle.

## Contenido del repositorio

- `opinion.txt` → Reflexión personal sobre el documental.
- `README.md` → Información general del trabajo.

## Objetivo

Analizar críticamente el impacto de las redes sociales en la sociedad y reflexionar sobre cómo estas plataformas influyen en los hábitos, comportamientos y decisiones de las personas.
